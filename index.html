Angry Ponies for Google Chrome

Angry Birds, but 20% cooler

Step 1: Install the Google Chrome extension.

Step 2: Play Angry Ponies!

Chubby pony graphics from PhantomDarkLover and WillDrawForFood. Code released to the public domain.

If you'd like to contribute artwork or sound design to make this more awesome (please do!), or would like to port the extension to a different browser, all instructions on how to contribute are at Github.
